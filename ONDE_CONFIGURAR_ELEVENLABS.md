# 📍 Onde Configurar ElevenLabs na Plataforma

## ❌ NÃO é na Seção "Ferramentas"

A seção **"Ferramentas"** é para outras integrações (webhooks, client tools, etc.), **NÃO** para ElevenLabs.

---

## ✅ O Local Correto: "Agente" → "Voz"

### **Como Acessar:**

1. **Faça login** na plataforma: https://www.sdrjuridico.com.br

2. **No menu lateral**, procure por:
   - **"Agente"** ou **"Agent"**
   - Clique nele

3. **Dentro de "Agente"**, procure por:
   - **"Voz"** ou **"Voice"**
   - Ou uma aba/seção chamada **"Configuração de Voz"**

4. **Você verá:**
   - Campo para **API Key do ElevenLabs**
   - Campo para **Voice ID**
   - Campo para **Voice Name**
   - Switch para **Ativar Voz**
   - Botão **"Testar Voz"**

## 🔍 Se Não Encontrar a Seção "Voz"

### **Opção 1: Verificar Menu**
- Procure por **"Agente"** no menu lateral
- Pode estar em **"Configurações"** → **"Agente"** → **"Voz"**

### **Opção 2: URL Direta**
Tente acessar diretamente:
```
https://www.sdrjuridico.com.br/agent/voice
```
ou
```
https://www.sdrjuridico.com.br/agent?tab=voice
```

### **Opção 3: Verificar Permissões**
- Certifique-se de estar logado
- Verifique se você tem permissão para acessar configurações de agente

---

## 📋 O Que Você Precisa Ter em Mãos

Antes de configurar, tenha pronto:

1. **API Key do ElevenLabs:**
   - Obtenha em: https://elevenlabs.io → Profile → API Keys
   - Formato: `sk_1234567890abcdef...`

2. **Voice ID:**
   - Escolha uma voz em: https://elevenlabs.io → Voices
   - Copie o Voice ID (ex: `21m00Tcm4TlvDq8ikWAM`)

3. **Voice Name:**
   - Nome da voz (ex: "Sarah")

## 🆘 Ainda Não Encontrou?

Se você não encontrar a seção "Voz":

1. **Verifique se está logado** corretamente
2. **Procure em "Configurações"** → pode estar lá
3. **Verifique permissões** - pode precisar de permissão de admin
4. **Entre em contato** se ainda não encontrar
