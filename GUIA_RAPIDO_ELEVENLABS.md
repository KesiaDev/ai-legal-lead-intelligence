# ⚡ Guia Rápido - ElevenLabs (5 minutos)

## 🎯 Configurar Voz no Plano Gratuito

### **1. Criar Conta (2 min)**
1. Acesse: https://elevenlabs.io
2. Clique em **"Sign Up"**
3. Crie conta (email ou Google)

### **2. Obter API Key (1 min)**
1. Perfil → **"API Keys"**
2. **"Create API Key"**
3. **Copie a API Key**

### **3. Escolher Voz (1 min)**
1. Menu → **"Voices"**
2. Escolha uma voz (ex: "Sarah")
3. **Copie o Voice ID**

### **4. Configurar na Plataforma (1 min)**
1. Acesse: https://www.sdrjuridico.com.br
2. **Agente** → **Voz**
3. Cole:
   - **API Key:** (cole aqui)
   - **Voice ID:** (cole aqui)
   - **Voice Name:** Sarah
4. **Ativar voz** ✅
5. **Salvar**

### **5. Testar**
1. Clique em **"Testar Voz"**
2. Digite: "Olá, como posso ajudar?"
3. Ouça o áudio ✅
