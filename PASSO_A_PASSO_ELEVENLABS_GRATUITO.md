# 🎤 Passo a Passo - Configurar ElevenLabs (Plano Gratuito)

## ✅ Resposta Rápida

**Sim, você pode usar o plano gratuito!** Ele inclui:
- ✅ 10.000 caracteres/mês (grátis)
- ✅ 3 vozes customizadas
- ✅ API Key gratuita
- ✅ Acesso à biblioteca de vozes

**É suficiente para testar!** 🎉

---

## 📋 Passo 1: Criar Conta no ElevenLabs (2 minutos)

### **1.1. Acessar Site**
1. Acesse: **https://elevenlabs.io**
2. Clique em **"Sign Up"** ou **"Get Started"**
3. Crie conta (pode usar email ou Google)

### **1.2. Confirmar Email**
1. Verifique seu email
2. Confirme a conta

---

## 📋 Passo 2: Obter API Key (1 minuto)

### **2.1. Gerar API Key**
1. Faça login no ElevenLabs
2. Clique no seu **perfil** (canto superior direito)
3. Vá em **"Profile"** → **"API Keys"**
4. Clique em **"Create API Key"**
5. Dê um nome (ex: "SDR Advogados")
6. **⚠️ COPIE A API KEY AGORA!** (ela só aparece uma vez!)

**Exemplo de API Key:**
```
sk_1234567890abcdefghijklmnopqrstuvwxyz
```

---

## 📋 Passo 3: Escolher Voz (1 minuto)

### **3.1. Acessar Biblioteca de Vozes**
1. No ElevenLabs, vá em **"Voices"** (menu lateral)
2. Explore as vozes disponíveis

### **3.2. Vozes Recomendadas (Gratuitas)**

**Vozes Femininas:**
- **Sarah** - Profissional, clara
- **Rachel** - Amigável, calorosa
- **Domi** - Jovem, energética

**Vozes Masculinas:**
- **Adam** - Profissional, confiável
- **Antoni** - Natural, conversacional
- **Arnold** - Autoritário, sério

### **3.3. Copiar Voice ID**
1. Clique na voz escolhida
2. **Copie o Voice ID** (ex: `21m00Tcm4TlvDq8ikWAM`)

**Onde encontrar:**
- Na URL da página da voz
- Ou no código da voz (ex: `EXAVITQu4vr4xnSDxMaL`)

---

## 📋 Passo 4: Configurar na Plataforma (2 minutos)

### **4.1. Acessar Configurações de Voz**
1. Acesse: **https://www.sdrjuridico.com.br**
2. Faça login
3. Vá em **"Agente"** → **"Voz"** (ou **"Configurações"** → **"Voz"**)

### **4.2. Preencher Dados**

**Campos Obrigatórios:**

1. **API Key do ElevenLabs:**
   - Cole a API Key que você copiou no Passo 2
   - Exemplo: `sk_1234567890abcdefghijklmnopqrstuvwxyz`

2. **Voice ID:**
   - Cole o Voice ID da voz escolhida
   - Exemplo: `21m00Tcm4TlvDq8ikWAM`

3. **Voice Name:**
   - Nome da voz (ex: "Sarah")

**Campos Opcionais (pode deixar padrão):**

- **Probabilidade de resposta em áudio:**
  - Em texto: "Nunca" (padrão) - economiza caracteres
  - Em áudio: "Alta" (padrão)
  - Em mídia: "Baixa" (padrão)

- **Parâmetros da voz:**
  - Estabilidade: 50% (padrão)
  - Similaridade: 75% (padrão)
  - Velocidade: 1.0x (padrão)

### **4.3. Ativar Voz**
1. Marque **"Ativar Voz"** ou **"Enabled"** (switch no topo)
2. Clique em **"Salvar"** ou **"Save"**

---

## 📋 Passo 5: Testar Voz (1 minuto)

### **5.1. Testar na Interface**
1. Na tela de configuração de voz
2. Clique em **"Testar Voz"** ou **"Test Voice"**
3. Digite um texto de teste (ex: "Olá, como posso ajudar?")
4. Clique em **"Gerar"** ou **"Generate"**
5. Ouça o áudio gerado ✅

### **5.2. Testar via WhatsApp**
1. Envie uma mensagem via WhatsApp para o número conectado
2. O agente deve responder em áudio (se configurado)
3. Verifique se o áudio foi gerado e enviado

## 💰 Fazer Upgrade para Pro (Depois)

### **Quando Fazer Upgrade?**

**Faça upgrade quando:**
- ✅ Você precisar de mais de 10.000 caracteres/mês
- ✅ Quiser vozes customizadas ilimitadas
- ✅ Quiser remover marca d'água
- ✅ Quiser prioridade no suporte

**Plano Gratuito é suficiente para:**
- ✅ Testes iniciais
- ✅ Pequenos volumes
- ✅ Validação do conceito

### **Como Fazer Upgrade:**

1. **Acesse:** https://elevenlabs.io
2. **Faça login**
3. **Clique no perfil** (canto superior direito)
4. **Vá em "Subscription"** ou **"Billing"**
5. **Escolha o plano:**
   - **Creator** ($5/mês) - 30.000 caracteres
   - **Pro** ($22/mês) - 100.000 caracteres
   - **Business** (Customizado) - Volume ilimitado
6. **Complete o pagamento**
7. **✅ Sua API Key continua a mesma!** (não precisa reconfigurar)

## 🆘 Problemas Comuns

### **Erro: "API Key inválida"**
- ✅ Verifique se copiou a API Key completa
- ✅ Certifique-se de que não há espaços extras
- ✅ Gere uma nova API Key se necessário

### **Erro: "Voice ID não encontrado"**
- ✅ Verifique se o Voice ID está correto
- ✅ Certifique-se de que a voz existe no ElevenLabs

### **Erro: "Limite de caracteres excedido"**
- ✅ Você usou os 10.000 caracteres do plano gratuito
- ✅ Faça upgrade para Pro ou aguarde o próximo mês
