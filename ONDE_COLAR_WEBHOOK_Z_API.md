# 📍 Onde Colar a URL do Webhook no Z-API

## ✅ Campo Correto: "Ao receber"

Na página **"Webhooks e configurações gerais"**, você verá vários campos. O campo que você precisa preencher é:

**"Ao receber"** (On receive) - Campo com ícone de setas duplas para a esquerda ⬅️⬅️

---

## 🎯 Passo a Passo Visual

### **1. Localizar o Campo**

Na página, procure por:
- **"Ao receber"** (On receive)
- Ícone: ⬅️⬅️ (setas duplas para a esquerda)
- Placeholder: "Ao receber"

### **2. Colar a URL**

No campo **"Ao receber"**, cole:

```
https://api.sdrjuridico.com.br/api/webhooks/zapi
```

### **3. Configurar Toggle (Opcional)**

Abaixo do campo "Ao receber", há um toggle:
- **"Notificar as enviadas por mim também"**
- Deixe **DESLIGADO** (OFF) - não queremos receber nossas próprias mensagens

### **4. Salvar**

Clique no botão verde **"Salvar"** no final da página

---

## 📋 Campos da Página (Para Referência)

Você verá vários campos, mas **só precisa preencher um:**

- ❌ **"Ao enviar"** - Não precisa (mensagens que enviamos)
- ❌ **"Ao desconectar"** - Não precisa
- ✅ **"Ao receber"** - **AQUI!** Cole a URL
- ❌ **"Presença do chat"** - Não precisa
- ❌ **"Receber status da mensagem"** - Não precisa
- ❌ **"Ao conectar"** - Não precisa

---

## 🔗 URL Completa para Colar

```
https://api.sdrjuridico.com.br/api/webhooks/zapi
```
