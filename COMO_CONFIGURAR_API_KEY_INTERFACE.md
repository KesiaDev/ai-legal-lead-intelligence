# ✅ Campo de API Key Adicionado na Interface!

## 🎉 O Que Foi Feito

Adicionei o campo **"API Key do ElevenLabs"** na interface de configuração de voz!

---

## 📍 Onde Está

**Agente IA** → **Voz** → **Configuração de Voz**

Agora você verá:
- ✅ Campo **"API Key do ElevenLabs"** (obrigatório)
- ✅ Campo **"Provider"** (ElevenLabs)
- ✅ Campo **"Voz"** (seleção de voz)
- ✅ Parâmetros da voz
- ✅ Regras de resposta

---

## 🎯 Como Usar

### **1. Obter API Key**
1. Acesse: https://elevenlabs.io
2. Perfil → **"API Keys"**
3. **"Create API Key"**
4. **Copie a API Key** (ela só aparece uma vez!)

### **2. Configurar na Plataforma**
1. Acesse: **Agente IA** → **Voz**
2. Preencha:
   - **API Key do ElevenLabs:** Cole a API Key
   - **Voz:** Escolha uma voz (ex: "Sarah")
3. **Ative a voz** (switch no topo)
4. **Salve** (salva automaticamente quando você preenche)

---

## ✅ Funcionalidades

- ✅ Campo de API Key adicionado
- ✅ Salva automaticamente no backend
- ✅ Carrega configuração do backend ao abrir
- ✅ Integrado com a API `/api/voice/config`
